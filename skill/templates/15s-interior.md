# Template: 15-Second Interior UGC
# Use for: videos entirely inside the shop

## Structure:
# 0-3s: Hook — she looks at product/item, says opening line
# 3-8s: Interaction with product — picks up, shows logo, holds to camera
# 8-12s: Eating/tasting moment — natural reaction
# 12-15s: CTA line + chef's kiss cutoff

## Notes:
# - Phone always in ONE hand (left preferred)
# - Other hand handles all products
# - Logo always faces camera — never reversed
# - Green baklava = NO crunch sound
# - Golden baklava = soft pastry crunch only
# - Full body visible in first shot, waist-up after
# - Chef's kiss: 3 fingers pinched → lips → hold → open outward
# - Ends abruptly — no fade
